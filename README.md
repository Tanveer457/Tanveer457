## 🚀 Current Research & Projects

### 🔎 Real-Time Industrial Fire Hazard Detection System *(FYP)*
An end-to-end computer vision pipeline engineered to detect anomalies in complex, low-visibility industrial environments. 

> **Key Contributions:**
> *   **Dataset Curation:** Engineered a highly specialized custom dataset of **5,000+ annotated images**, focusing on edge-case ambient conditions (smoke obfuscation, low lighting).
> *   **Model Architecture:** Trained and fine-tuned **YOLOv12n** using advanced data augmentation strategies (via Albumentations) to prevent overfitting.
> *   **Edge Optimization:** Deployed the inference pipeline to an **NVIDIA Jetson Nano**, maximizing Frames Per Second (FPS) while preserving critical Mean Average Precision (mAP) boundaries for life-safety applications.
